# Legged Model Zoo

An incremental MATLAB R2019b-compatible framework for hybrid legged models. Run `startup`, then discover declarative model manifests with `lmz.registry.ModelRegistry.discover()`.

The current reconstruction implements the core schema/chart, run-control, registry, artifact, and quadruped legacy-layout boundaries. Numerical equivalence is not yet claimed; see `MIGRATION_STATUS.md` and `docs/TEST_STATUS.md`.
